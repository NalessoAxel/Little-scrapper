# Little-scrapper

Make sure you have nodemon install on your computer.
`npm i nodemon -g`

`npm i` then `npm run` to run the project on the local server.

You can replace the url by the url of your choice.


If you get `Request failed with status code 403` you'r are not authorize to scrap the website.

You can see this in your browser console in the section "Network" choose "Fetch/XHR" and find the fetch where you can see `{toScrap: false} toScrap: false`
